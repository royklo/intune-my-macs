# Changelog

All notable changes to **Intune my Macs** are documented in this file.

The format is loosely based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/). Dates are `YYYY-MM-DD`. Newest entries are at the top.

> This project is a **proof of concept** and does not follow semantic versioning. Changes are recorded by date rather than release tag.

| 2026-07-13 | **Added** SCR-APP-108 OneDrive Open at Login; **Removed** the deprecated OpenAtLogin key from POL-APP-100 | The OneDrive `OpenAtLogin` managed preference is deprecated since sync app 24.113 and a no-op on current builds, so it was removed from [`macOS/configurations/office/pol-app-100-office.json`](macOS/configurations/office/pol-app-100-office.json) (setting IDs renumbered, `settingCount` now 21, manifest bumped to 2.1). New user-context script [`scr-app-108-onedrive-openatlogin.zsh`](macOS/scripts/intune/scr-app-108-onedrive-openatlogin.zsh) with manifest [`scr-app-108-onedrive-openatlogin.xml`](macOS/scripts/intune/scr-app-108-onedrive-openatlogin.xml) replaces it: it enables the OneDrive login item via Microsoft's `/createloginitem` command (OneDrive 26.027+), the only mechanism that actually enables (not just allows) the login item. Guarded and idempotent - runs every 15 minutes, no-ops until OneDrive is installed and new enough, performs the enable once, then a per-user marker keeps every later run a no-op. | Roy Klooster |
| 2026-07-13 | **Fixed** doc generator after `macOS/` restructure | Fixed [`tools/Generate-ConfigurationDocumentation.py`](tools/Generate-ConfigurationDocumentation.py), which had produced an empty document since the `macOS/` restructure: updated stale glob paths, accepted the `IntuneManifest` manifest root alongside `MacIntuneManifest`, and resolved platform-relative `SourceFile` paths (also removing a duplicated `pol-sys-105` section). Updated all references: [`agency.toml`](agency.toml), [`AGENTS.md`](AGENTS.md), [`docs/architecture.md`](docs/architecture.md), [`docs/conventions.md`](docs/conventions.md), [`docs/testing-patterns.md`](docs/testing-patterns.md), [`.github/copilot-instructions.md`](.github/copilot-instructions.md), [`.github/PULL_REQUEST_TEMPLATE.md`](.github/PULL_REQUEST_TEMPLATE.md), [`.github/prompts/pr-review-reply.prompt.md`](.github/prompts/pr-review-reply.prompt.md), and [`.github/workflows/copilot-setup-steps.yml`](.github/workflows/copilot-setup-steps.yml). | Neil Johnson |
